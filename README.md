# GYM Final Project

Full-stack gym management project.

## Structure
- `backend/` Laravel API
- `frontend/` React (Vite)

## Local Dev
Backend:
```pwsh
cd backend
php artisan serve --host 127.0.0.1 --port 8000
```
Frontend:
```pwsh
cd frontend
npm run dev -- --host 127.0.0.1 --port 5174
```

Open:
- Frontend: http://127.0.0.1:5174/
- Backend: http://127.0.0.1:8000/
